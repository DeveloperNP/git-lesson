GIT commands:
  • git config --global user.name "Name" - глобальная настройка имени пользователя
  • git config --global user.email "Email" - глобальная настройка почты пользователя
  • git clone path-to-repository - склонировать репотизиторий к себе
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить новый в файл под контроль GIT-а
  • git commit -am "comment" - фиксация изменений с комментарием
  • git push - отправляет файлы на сервер
  • git pull - позволяет забрать измененные файлы с сервера к себе в локальный репотизиторий
